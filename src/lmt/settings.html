Filenames

Use Custom Filenames

Filenames will be the playlist name in the URL.
Filenames will be created using the template below:
  • %%userid%% User ID
  • %%username%% Username
  • %%videoid%% Video ID
  • %%videotitle%% Video Title
  • %%videotime%% Video Time

The filename of the playlist will be used if the custom title ends up being empty.
